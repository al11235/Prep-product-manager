# Learning Logs

Each user has their own learning log file in this directory, named `{Name}_learning_logs.md`.

## Current Users
- `Alok_learning_logs.md` — Alok's personal progress tracker

## Adding a New User
When a new person starts using this repo:
1. Create a new file: `{Name}_learning_logs.md`
2. Use the template from `INSTRUCTIONS.md` in the root
3. The AI coach will update this file after each practice session

## How It Works
- Each person's gaps, mistakes, and progress are tracked separately
- The AI reads YOUR log at the start of each session to target your weak spots
- Shared resources (questions, frameworks, plan) stay the same for everyone
- Only your personal log gets updated during practice
