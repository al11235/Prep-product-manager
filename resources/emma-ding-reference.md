## Course Structure: Product Case Interview Mastery

### Module 0: Start Here
- What is a product case interview

### Module 1: Measure Success
**Lessons (from course listing):**
- How to Define a Metric (10:36)
- Engagement Metrics (9:36)
- Customer Lifetime Value Pt 1 (12:18)
- Customer Lifetime Value Pt 2 (9:19)
- Customer Churn (14:20)
- Interview Q&A: Measure Success (9:06)
- Interview Q&A: Measure Success — Common Mistakes (3:23)
- Facebook: Pros and Cons of a Metric (16:23)
- DoorDash: Measure Customers' Satisfaction (14:59)

**Key concepts:**
- Defining the right metric for a product/feature
- Understanding engagement metrics (DAU, MAU, sessions, time spent)
- Customer Lifetime Value (CLV) — what it is, how to estimate it
- Customer churn — measuring and predicting it
- Pros/cons of common metrics (e.g., DAU can be gamed)
- AARRR funnel metrics (Acquisition, Activation, Retention, Revenue, Referral)

### Module 2: A/B Testing
**Key concepts (from blog + course):**
- What is an A/B test and when to use one
- Selecting metrics for experimentation
- Choosing randomization units (user, session, device, geo)
- Determining target population
- Computing sample size (power analysis)
- Determining test duration
- Analyzing results (significance, practical significance)
- Common pitfalls (peeking, novelty, Simpson's)
- When you CAN'T A/B test — alternatives

**7 A/B Testing Questions (from her Towards Data Science article):**
1. What is an A/B test?
2. How do you choose the right metric?
3. How do you determine sample size?
4. How long should you run a test?
5. What if the results are significant for one metric but not another?
6. What are common mistakes in A/B testing?
7. What do you do when you can't run an A/B test?

### Module 3: Diagnose a Problem
**Key concepts (from blog):**
- Framework for diagnosing a metric drop/change
- Step 1: Clarify the definition of the metric
- Step 2: Determine temporal aspect (sudden vs. gradual)
- Step 3: Check if other products/features show the same change
- Step 4: Segment users by demographics and behavioral features to isolate
- Step 5: Form hypotheses and validate
- Step 6: Recommend action

**Example questions:**
- "Facebook group creation dropped 20%. What's going on?"
- "Time spent on the app decreased. Investigate."
- "Notification open rates dropped 30%. Diagnose."

### Module 4: Improve a Product
**Key concepts:**
- How to structure a "how would you improve X?" answer
- Identifying user pain points from data
- Prioritizing improvements (impact vs. effort)
- Proposing how to validate improvements (experimentation)
- Connecting improvements to metrics

**Example questions:**
- "How would you improve Facebook Marketplace?"
- "How would you use data to improve the Instagram Explore page?"

### Module 5: Make Strategic Decisions
**Key concepts:**
- Launch or not launch decisions
- Trade-off analysis (conflicting metrics)
- Cost-benefit reasoning
- Short-term vs. long-term framing
- Recommending with incomplete information

**Example questions:**
- "An A/B test shows CTR is up but revenue is down. Should we launch?"
- "Should we expand to a new market given these metrics?"
- "Two features compete for the same surface. Which do you ship?"

### Module 6: Product-Specific Questions
**Key concepts:**
- Company-specific product knowledge (Meta, Google, Amazon, etc.)
- Understanding business models and how they affect metric choices
- Tailoring answers to the specific product context

## Emma Ding's 7 Types of Product Case Questions

From her blog: [How to Ace the 7 Types of Product Case Interview Questions](https://www.emmading.com/blog/how-to-ace-the-7-types-of-product-case-interview-questions)

| # | Type | Example |
|---|------|---------|
| 1 | Measure Success | "How would you measure success for Instagram Reels?" |
| 2 | Diagnose a Problem | "Time spent on app decreased 10%. What happened?" |
| 3 | A/B Testing / Experiment Design | "How would you design an experiment to test a new feature?" |
| 4 | Improve a Product | "How would you improve Facebook Marketplace?" |
| 5 | Launch or Not | "A/B test shows CTR up but satisfaction down. Launch?" |
| 6 | Conflicting Metrics | "Engagement up but revenue down. What do you do?" |
| 7 | Pros/Cons of a Metric | "What are the pros and cons of using DAU as a success metric?" |
